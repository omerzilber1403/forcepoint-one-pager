# Color Palette

## Primary Palette

| Role | Hex | Notes |
|---|---|---|
| Page background | `#0a0a0f` | Near-black with blue tint |
| Card surface | `#12121a` | Slightly lighter than bg |
| Subtle border | `#1e1e2e` | Barely visible dividers |
| Accent Indigo | `#6366f1` | Tailwind `indigo-500` — primary brand |
| Accent Cyan | `#22d3ee` | Tailwind `cyan-400` — metrics, secondary |

## Text

| Role | Hex | Tailwind Equivalent |
|---|---|---|
| Primary text | `#f8fafc` | `slate-50` |
| Secondary text | `#94a3b8` | `slate-400` |
| Muted text | `#475569` | `slate-600` |
